# bass

> Scale-locked, clock-synced grid bass instrument

## Controls

- **E1** — tempo (BPM)
- **E2** — note length / subdivision
- **E3** — volume
- **K2** — toggle play/stop
- **K3** — clear sequence

## Grid

8×16 layout: rows 1–6 are note pads (columns = scale degrees, rows = octaves); row 7 selects root note; row 8 selects scale, octave offset, and transport.

## Requirements

- norns
- grid (16x8 or larger)

## Install

```
;install https://github.com/jamminstein/bass
```
